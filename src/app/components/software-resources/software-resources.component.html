Software Resources


VCell https://VCell.org
Spatial and nonspatial deterministic, stochastic, hybrid stochastic/deterministic simulations with GUI and central database and computing services
COPASI https://copasi.org
Nonspatial deterministic and stochastic, parameter optimization and steady state analysis.
SpringSaLaD https://vcell.org/ssalad
Particle-based, stochastic, biochemical simulation platform for meso-scale modeling that explicitly accounts for individual domains within a protein.
Amici https://amici.readthedocs.io/en/latest/index.html
A multi-language interface to the SUNDIALS solvers CVODES and IDAS.
Berkeley Madonna https://berkeley-madonna.myshopify.com/
General purpose differential equations solver with GUI for constructing complex mathematical models.
BioDynaMo https://biodynamo.org/
Platform for 3D agent-based simulations.
BioNetGen/RuleBender https://bionetgen.org/
Rule-based dynamical modeling; RuleBender is a GUI interface for BioNetGen.
BoolNet https://cran.r-project.org/package=BoolNet
R package for the construction and analysis of synchronous, asynchronous, probabilistic and temporal Boolean networks.
Cell Collective https://cellcollective.org/
Web-based platform for the construction, simulation, and analysis of Boolean-based models.
Cell Designer http://www.celldesigner.org/
A structured diagram editor for drawing gene-regulatory and biochemical networks stored in SBKL.
CellNetAnalyzer http://www2.mpi-magdeburg.mpg.de/projects/cna/cna.html
MATLAB toolbox for exploring structural and functional properties of metabolic, signaling, and regulatory networks using stoichiometric and constraint-based techniques as well as Boolean and multivalued logic.
CellNOpt https://saezlab.github.io/CellNOptR/
Creates logic-based models of signal transduction networks using different logic formalisms (Boolean, Fuzzy, or differential equations). Encodes information on signaling pathways as a Prior Knowledge Network, and trains it against high-throughput biochemical data to create cell-specific models.
CobraPy https://opencobra.github.io/cobrapy/
Python package that provides a simple interface to metabolic constraint-based reconstruction and analysis.
CNApy https://cnapy-org.github.io/CNApy/
A cross-platform desktop application written in Python for constraint-based analysis of metabolic networks.
CoMSES https://www.comses.net/resources/modeling-frameworks/
Compilation of software for agent- based modeling.
EpiLog http://epilog-tool.org/
Models of simple epithelial are defined as cellular automata, where the behavior of each cell is governed by its (logical) regulatory model, subject to input signals from neighboring cells.
E-Cell System https://ecell4.e-cell.org/
A platform for modeling, simulation and analysis of complex, heterogeneous and multi-scale systems.
Escher-FBA https://sbrg.github.io/escher-fba/#/
An interactive pathway visualization tool for on-the-fly flux balance analysis (FBA) calculations.
GFRD https://gfrd.org/
Algorithm for single particle reaction diffusion systems using Green's Function.
GillesPy2 https://github.com/StochSS/GillesPy2
Python package for stochastic simulation of biochemical systems.
GINsim http://www.ginsim.org/home
Specifies a model of a genetic regulatory network in terms of asynchronous, multivalued logical functions, and simulates and/or analyses its qualitative dynamical behavior.
Ibiosim https://async.ece.utah.edu/tools/ibiosim/
Modeling, analysis, and design of genetic circuits
JWSonline https://jjj.biochem.sun.ac.za/
Tool for the construction, modification and simulation of kinetic models and for the storage of curated models.
MaBoSS https://maboss.curie.fr/
Markovian Boolean Stochastic Simulator for simulating continuous/discrete time Markov processes based on Boolean networkss based on Boolean networks.
MCell https://mcell.org/
Monte Carlo spatial stochastic reaction diffusion simulator
modelbase https://pypi.org/project/modelbase/
Python package for building and analyzing dynamic mathematical models originally designed for simulating metabolic systems.
Morpheus http://morpheus.gitlab.io/
Modeling and simulation environment for the study of multi-scale and multicellular systems using ODEs, PDEs and cellular Potts models.
NERDSS https://github.com/mjohn218/NERDSS
NonEquilibrium Reaction-Diffusion Self-assembly Simulator
Neuron https://www.neuron.yale.edu/neuron/
Environment for building and using computational models of neurons and networks of neurons.
OpenCOR https://opencor.ws/
An open source cross-platform modelling environment
PySCeS http://pysces.sourceforge.net/
A collection of Python-based tools for the analysis of cellular systems.
Simmune https://www.niaid.nih.gov/research/simmune-project
Spatial modeling of cell biological systems from interactions between molecular domains to the behavior of populations of cells
Smoldyn http://www.smoldyn.org/
Spatial Stochastic simulator for chemical reaction networks.
Spatiocyte https://spatiocyte.org/
Lattice-based particle simulator
StochSS https://stochss.org/
Web interface for simulating stochastic and ODE-based biochemical systems.
Tellurium http://tellurium.analogmachine.org/
Python-based modeling environment for systems and synthetic biology.
XPP http://www.math.pitt.edu/~bard/xpp/xpp.html
A tool for solving differential, difference, delay, functional and stoachastic equations as well as boundary value problems.
