CompCellBio


Computational Cell Biology


About: The CompCellBio program is funded as a NIGMS research resource for mathematical modeling in cell biology. We provide a gateway to resources for modeling including software, solvers, databases, compute resources and training for cell biologists . This site specifically supports the following software



COPASI for simulation and analysis of biochemical networks and their dynamics.


VCell a comprehensive platform for modeling cell biological systems using nonspatial (ODE or stochastic ) or spatial (PDE or stochastic) methods.


SpringSaLaD  for explicitly modeling binding events and state changes among multivalent molecules using Langevin Dynamics.
